# ARM 汇编基础

## 1. GNU 汇编语法

在使用 GCC 交叉编译器编写 ARM 汇编时，需要遵循 GNU 汇编语法。GNU 汇编文件通常由一行一条语句组成，基本格式如下：

```asm
label: instruction @ comment
```

- `label`：标号，表示地址位置，可用于跳转或表示数据地址。
- `instruction`：指令或伪指令。
- `comment`：注释内容，常用 `@`，也可以使用 `/* ... */`。

示例：

```asm
add:
    movs r0, #0x12    @ r0 = 0x12
```

### 1.1 基本规则

- 指令、伪指令、寄存器名可以全部大写，也可以全部小写，但不要混用。
- 以 `:` 结尾的标识符会被识别为标号。
- `_start` 常作为默认入口标号，也可以在链接脚本中通过 `ENTRY` 指定其他入口。

### 1.2 常见段

可以使用 `.section` 定义段，也可以直接使用 GNU 汇编预定义的常见段名。

| 段名 | 说明 |
| --- | --- |
| `.text` | 代码段 |
| `.data` | 已初始化数据段 |
| `.bss` | 未初始化数据段 |
| `.rodata` | 只读数据段 |

示例：

```asm
.section .testsection
```

### 1.3 常见伪操作

| 伪操作 | 说明 |
| --- | --- |
| `.byte` | 定义单字节数据 |
| `.short` | 定义半字数据 |
| `.long` | 定义 4 字节数据 |
| `.equ` | 定义常量 |
| `.align` | 数据对齐 |
| `.end` | 源文件结束 |
| `.global` | 定义全局符号 |

示例：

```asm
.global _start
.equ num, 0x12
.align 4
```

## 2. 汇编函数

ARM GNU 汇编中的函数通常以标号开头，函数体结束后使用 `bx lr` 返回。

```asm
my_func:
    mov r0, r1
    bx lr
```

说明：

- `bx lr` 是最常见的返回方式。
- 如果函数不需要返回，也可以直接跳转到其他代码路径。

## 3. 常用指令

### 3.1 数据传输指令

常用的数据传输指令有 `mov`、`mrs` 和 `msr`。

| 指令 | 作用 |
| --- | --- |
| `mov` | 在通用寄存器之间传递数据，或将立即数加载到寄存器 |
| `mrs` | 读取特殊寄存器到通用寄存器 |
| `msr` | 将通用寄存器的数据写入特殊寄存器 |

示例：

```asm
mov r0, r1        @ r0 = r1
mov r0, #0x12     @ r0 = 0x12
mrs r0, cpsr      @ r0 = cpsr
msr cpsr, r0      @ cpsr = r0
```

### 3.2 存储器访问指令

ARM 在访问外设寄存器或内存数据时，通常使用 `ldr` 和 `str`。

| 指令 | 作用 |
| --- | --- |
| `ldr rd, [rn, #offset]` | 从地址 `rn + offset` 读取数据到 `rd` |
| `str rd, [rn, #offset]` | 将 `rd` 中的数据写入地址 `rn + offset` |

说明：

- `ldr` 还可以用来加载立即数，这时要写成 `ldr r0, =0x12345678`，不是 `#`。
- `ldr` 和 `str` 默认按字访问；按字节和半字访问时可使用 `ldrb`、`strb`、`ldrh`、`strh`。

示例：

```asm
ldr r0, =0x0209c004   @ 寄存器地址
ldr r1, [r0]          @ 读取地址中的值

ldr r0, =0x0209c004   @ 寄存器地址
ldr r1, =0x20000002   @ 要写入的值
str r1, [r0]          @ 写入地址
```

### 3.3 压栈和出栈指令

在函数调用或中断处理中，常常需要先保存现场，再恢复现场。常用指令是 `push` 和 `pop`。

| 指令 | 作用 |
| --- | --- |
| `push {reg list}` | 将寄存器列表压入栈中 |
| `pop {reg list}` | 从栈中恢复寄存器列表 |

示例：

```asm
push {r0-r3, r12, lr}

@ 这里执行函数或中断处理

pop {r0-r3, r12, lr}
```

`push` 和 `pop` 也可以看作 `stmfd sp!` 和 `ldmfd sp!` 的常用写法。

```asm
stmfd sp!, {r0-r3, r12, lr}
ldmfd sp!, {r0-r3, r12, lr}
```

### 3.4 跳转指令

常见跳转方式有 `b`、`bl`、`bx` 和 `blx`。

| 指令 | 作用 |
| --- | --- |
| `b <label>` | 直接跳转到标号 |
| `bl <label>` | 跳转到标号，并把返回地址保存到 `lr` |
| `bx <rm>` | 跳转到寄存器指定地址，并切换指令集状态 |
| `blx <rm>` | 结合 `bl` 和 `bx` 的特点，支持间接调用和指令集切换 |

说明：

- `b` 适合不需要返回的跳转。
- `bl` 适合函数调用。
- `bx lr` 常用于函数返回。

示例：

```asm
_start:
    ldr sp, =0x80200000
    b main
```

如果需要从汇编调用 C 函数，通常使用 `bl`：

```asm
bl system_irqhandler
```

### 3.5 算术运算指令

常用的算术运算指令如下。

| 指令 | 作用 |
| --- | --- |
| `add` | 加法 |
| `adc` | 带进位加法 |
| `sub` | 减法 |
| `sbc` | 带借位减法 |
| `mul` | 乘法 |
| `udiv` | 无符号除法 |
| `sdiv` | 有符号除法 |

示例：

```asm
add r0, r1, r2
sub r0, r1, #1
mul r0, r1, r2
```

### 3.6 逻辑运算指令

常用的逻辑运算指令如下。

| 指令 | 作用 |
| --- | --- |
| `and` | 按位与 |
| `orr` | 按位或 |
| `bic` | 位清除 |
| `eor` | 按位异或 |
| `orn` | 按位或非 |

示例：

```asm
and r0, r1, r2
orr r0, r1, #0x1
bic r0, r1, #0x3
eor r0, r1, r2
```
